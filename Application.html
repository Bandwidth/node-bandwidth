Create a new application

Parameters:
Name Type Attributes Description
params Object

Parameters for creating a new call

Properties
Name Type Attributes Description
name String <optional>

A name you choose for this application.
incomingCallUrl String <optional>

A URL where call events will be sent for an inbound call. This is the endpoint where the Application Platform will send all call events. Either incomingCallUrl or incomingMessageUrl is required.
incomingCallUrlCallbackTimeout String <optional>

Determine how long should the platform wait for incomingCallUrl's response before timing out in milliseconds.
incomingCallFallbackUrl String <optional>

The URL used to send the callback event if the request to incomingCallUrl fails.
incomingMessageUrl String <optional>

A URL where message events will be sent for an inbound message. This is the endpoint where the Application Platform will send all message events. Either incomingMessageUrl or incomingCallUrl is required.
incomingMessageUrlCallbackTimeout Number <optional>

Determine how long should the platform wait for incomingMessageUrl's response before timing out in milliseconds.
incomingMessageFallbackUrl String <optional>

The URL used to send the callback event if the request to incomingMessageUrl fails.
callbackHttpMethod String <optional>

Determine if the callback event should be sent via HTTP GET or HTTP POST. Values are "get" or "post", default: "post".
autoAnswer Boolean <optional>

Determines whether or not an incoming call should be automatically answered. Default value is 'true'.
callback function <optional>

A callback for the list of applications
