    playAudioAdvanced(conferenceId, params, callbackopt) → {Promise}

    Play audio file or speak sentence in conference

    Source:
    • conference.js, line 237
    Parameters:
    Name Type Attributes Description
    conferenceId String

    The ID of the conference
    params Object

    Parameters to play audio in conference.

    Name Type Attributes Default Description
    fileUrl String <optional>

    The http location of an audio file to play (WAV and MP3 supported).
    sentence String <optional>

    The sentence to speak.
    gender String <optional>
    		 female

    The gender of the voice used to synthesize the sentence. It will be considered only if sentence is not null. The female gender will be used by default.
    locale String <optional>
    		 en_US

    The locale used to get the accent of the voice used to synthesize the sentence. It will be considered only if sentence is not null/empty. The en_US will be used by default.
    voice String <optional>
    		 Susan

    The voice to speak the sentence. for list of supported voices It will be considered only if sentence is not null/empty. Susan's voice will be used by default.
    loopEnabled Boolean <optional>
    		 false

    When value is true, the audio will keep playing in a loop. Default: false.
    callback function <optional>

    Callback for the operation
    Returns:
    Type:
    Promise

    A promise for the operation

    Examples
    //Play Audio File on loop
var options = {
	fileUrl     : "http://myurl.com/file.mp3",
	loopEnabled : true
}
//Promise
client.Conference.playAudioAdvanced("conferenceId", options).then(function (res) {});

//Callback
client.Conference.playAudioAdvanced("conferenceId", options, function (err,res) {});
    //Speak sentence with options
var options = {
	sentence : "hola de Bandwidth",
	gender   : "male",
	locale   : "es",
	voice    : "Jorge"
}
//Promise
client.Conference.playAudioAdvanced("conferenceId", options).then(function (res) {});

//Callback
client.Conference.playAudioAdvanced("conferenceId", options, function (err,res) {});
