    The ID of the call
    params Object

    Parameters for creating a gather

    Name Type Attributes Default Description
    maxDigits String <optional>

    The maximum number of digits to collect (max: 30)
    interDigitTimeout String <optional>
    		 5

    Stop gathering if a DTMF digit is not detected in this many seconds (max: 30s)
    terminatingDigits String <optional>
    		 #

    A string of DTMF digits that end the gather operation immediately if any one of them is detected
    tag String <optional>

    A string you choose that will be included with the response and events for this gather operation.
    prompt.sentence String <optional>

    The text to speak for the prompt. Uses the same defaults as call.playAudioAdvanced. See the docs
    prompt.gender String <optional>

    The gender to use for the voice reading the prompt sentence
    prompt.locale String <optional>

    The language and region to use for the voice reading the prompt sentence
    prompt.loopEnabled Boolean <optional>
    		 false

    When value is true, the audio will keep playing in a loop
    prompt.bargeable Boolean <optional>
    		 true

    Make the prompt (audio or sentence) bargeable (will be interrupted at first digit gathered).
    prompt.fileUrl Strings <optional>

    Make the prompt (audio or sentence) bargeable (will be interrupted at first digit gathered).
    callback function <optional>

    Callback with the newly created call
    Returns:
    Type:
    CallResponse

    A promise for the newly created call

    Example
    //Create Gather
//The gather ends if either 0, #, or * is detected
var options = {
	maxDigits         : 30,
	interDigitTimeout : "30",
	terminatingDigits : "0#*",
	prompt            : {
		sentence    : "Please enter your account number and press pound",
		gender      : "male",
		voice       : "Simon",
		locale      : "en_UK",
		loopEnabled : true,
		bargeable   : true
	}
};
//Promise
client.Call.createGather("callId", options).then(function(res) {});

//Callback
