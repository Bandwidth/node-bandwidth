create(params, callbackopt) → {BridgeResponse}

Create a new bridge

Parameters:
Name Type Attributes Description
params Object

Parameters for creating a bridge

Properties
Name Type Attributes Default Description
bridgeAudio Boolean <optional>
 true

Enable/Disable two way audio path.
callIds Array.<String> <optional>

The list of call ids in the bridge. If the list of call ids is not provided the bridge is logically created and it can be used to place calls later.
callback function <optional>

Callback with the newly created bridge
Source:
  • bridge.js, line 40
Returns:

A promise for the newly created bridge

Type
BridgeResponse
Example
//Promise
client.Bridge.create({
	bridgeAudio: true,
	callIds: ['c-qbs5kwrsyx6wsdi', 'c-zan4g74pprsq']
})
.then(function (response) {
	console.log(response);
});

//Callback
client.Bridge.create({
	bridgeAudio: true,
	callIds: ['c-qbsx6wsdi', 'c-zan4g7prsq']
}, function (err, response) {
		if(err) {
			console.log(err);
		}
		else {
			console.log(response);
		}
	});

playAudioAdvanced(bridgeId, params, callbackopt) → {Promise}

Play audio file or speak sentence in bridge

Parameters:
Name Type Attributes Description
bridgeId String

The ID of the bridge
params Object

Parameters to play audio in bridge.

Properties
Name Type Attributes Default Description
fileUrl String <optional>

The http location of an audio file to play (WAV and MP3 supported).
sentence String <optional>

The sentence to speak.
gender String <optional>
 female

The gender of the voice used to synthesize the sentence. It will be considered only if sentence is not null. The female gender will be used by default.
locale String <optional>
 en_US

The locale used to get the accent of the voice used to synthesize the sentence. Check out docs for list of supported locales. It will be considered only if sentence is not null/empty. The en_US will be used by default.
voice String <optional>
 Susan

The voice to speak the sentence. Check out docs for list of supported voices It will be considered only if sentence is not null/empty. Susan's voice will be used by default.
loopEnabled Boolean <optional>
 false

When value is true, the audio will keep playing in a loop. Default: false.
callback function <optional>

Callback for the operation
Source:
  • bridge.js, line 276
Returns:

A promise for the operation

Type
Promise
Examples
//Play Audio File on loop
var options = {
	fileUrl     : "http://myurl.com/file.mp3",
	loopEnabled : true
}
//Promise
client.Bridge.playAudioAdvanced("bridgeId", options).then(function (res) {});

//Callback
client.Bridge.playAudioAdvanced("bridgeId", options, function (err,res) {});
//Speak sentence with options

	sentence : "hola de Bandwidth",
	gender   : "male",
	locale   : "es",
	voice    : "Jorge"
}
//Promise
